# This Web UI are created using Streamlit framework. The user can upload a audio file into the web UI and it converts the audio to text. The following are the source code and its dependencies

* Streamlit → Web UI
* SpeechRecognition → Speech-to-Text (simple + beginner-friendly)
* pydub - pydub is a simple audio processing library for Python. 
* ffmpeg → Handle mp3 / m4a / wav
* logging → Debugging & stability
